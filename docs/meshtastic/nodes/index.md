# Meshtastic Nodes

Коллекция реализованных нод Meshtastic с описанием характеристик, компонентов и опыта использования.

## Список нод

### [Astronomikon v1](astronomikon_v1/)

Автономная нода для установки на открытом воздухе со сменными аккумуляторами.

- **Тип установки**: Открытый воздух, стационарная
- **Питание**: 8x 18650 (сменные аккумуляторы)
- **Корпус**: IP65
- **Антенна**: Всенаправленная уличная AX-867R
- **Автономность**: ~11 дней на морозе

### [Костлявый v1](kostlyavy_v1/)

Первая экспериментальная нода с автономным питанием.

### [Костлявый v2](kostlyavy_v2/)

Эксперимент с дендрофекальными технологиями. Тестирование направленной антенны и солнечной батареи.

- **Питание**: 4x 18650 + солнечная батарея
- **Антенна**: Направленная волновой канал Антэкс AX-813Y
- **Мощность**: 20 дБм
- **Расстояние**: 47 км до города
- **Автономность**: 7-8 дней работы

### [Кукуруза v1](kukuruza_v1/)

Первая нода для установки на открытом воздухе со стационарным питанием.

- **Тип установки**: Открытый воздух, стационарная
- **Питание**: Стационарное (12V)
- **Корпус**: IP65
- **Антенна**: Всенаправленная уличная AX-867R
