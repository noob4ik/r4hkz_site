# Костлявый - эксперимент 2

Тут первый эексперимент с дендрофекальными технологиями

## Условия
- нода с автономным питанием - 4x 18650 
- попробовал дополнительно маленькую солнечную батарейку
- Протестировал [антенна волновой-канал Антэкс AX-813Y](https://www.ozon.ru/product/napravlennaya-4g-lte800-antenna-volnovoy-kanal-anteks-ax-813y-2088587894/)
- мощность 20дбм
- расстояние до города 47км

## Результаты:
- есть связь с городом
- заряда хватило на 7-8 дней работы (далее нода периодически появлялась в эфире - видимо немного работала зарядка)
